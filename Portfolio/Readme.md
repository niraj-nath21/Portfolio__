# Trial Portfolio

This repository contains a simple portfolio website created as a trial project. The purpose of this portfolio is to experiment with web design, layout, and technology choices before building a more advanced and polished personal portfolio in the future.

## Features
- Modern, responsive design using a purple color palette
- Introduction and About Me sections
- Social links to Instagram, LinkedIn, and GitHub
- Profile image
- Skills section (Django, Django REST framework, HTML, CSS, JavaScript, Node.js) with icons
- Interests section

## Note
This portfolio is just for trial and learning purposes. A more complete and professional portfolio will be developed later.

Feel free to explore, fork, or use this as a starting point for your own experiments!
